# Project Skill Registry

## Compact Rules

### Next.js 16/React 19
- Use App Router patterns.
- Be careful with breaking changes in Next.js 16 (React 19 support, etc.).
- Prefer Server Components where possible.
- Use `unstable_cache` for server-side data fetching revalidation.

### Tailwind CSS 4
- Use `@tailwindcss/postcss`.
- Modern Tailwind 4 syntax.
